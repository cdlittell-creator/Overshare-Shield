CLC Capstone • Web Application

Privacy awareness for social sharing

Overshare Shield is a privacy awareness web application designed to help users recognize when shared content may expose sensitive personal information. The project focuses on cybersecurity awareness, user consent, and privacy-first design.

Vision

What Overshare Shield is ultimately designed to achieve.

Our vision is to empower everyday users to make safer online choices by turning privacy risks into clear, understandable feedback. Overshare Shield aims to promote long-term cybersecurity awareness, strengthen digital responsibility, and make privacy protection more accessible to non-technical users.

Problem

What we are solving.

Social platforms make it easy to share quickly, but many users do not realize how personal details such as contact information, identifiers, or location indicators can be collected and misused. Oversharing increases risk for phishing, identity theft, stalking, and social engineering.

Methodology

How Overshare Shield works at a high level.

  1. User submits content for analysis.
  2. Text is extracted from submitted text or images.
  3. Personal identifiers are detected and classified.
  4. Weighted scoring assigns higher scores to more sensitive items like phone numbers and emails.
  5. Risk level output converts the score into Low, Medium, or High.
  6. User control supports exporting results and deleting data.

Tech Stack

Core technologies used to build and support the application.

Frontend

React and JavaScript power the user interface and interactive dashboard experience.

Backend

FastAPI and Python manage processing logic, analysis, and communication between system components.

Data Flow

JSON and multipart file uploads support communication between frontend input and backend analysis.

Image Processing

Pillow is used for image handling, metadata review, and privacy-related image sanitation features.

Validation

Pydantic supports structured validation and helps maintain clean, secure processing logic.

Deployment Style

Overshare Shield is designed as a local-first web application to maximize privacy and reduce unnecessary exposure of user data.

Privacy by Design

What we do to protect users from the start.

User Initiated Only

No background monitoring. The user decides what gets analyzed.

Data Minimization

Only necessary content is processed to generate privacy risk feedback.

User Control

Users can export results and delete analysis data.

Consent to Analyze Content

Overshare Shield analyzes content you voluntarily submit to identify potential privacy risks. No automatic data collection occurs. All analysis is initiated by you.

By continuing, you acknowledge:

  • You are submitting content voluntarily for privacy analysis.
  • Only the content you provide will be processed.
  • You may delete or export analysis results at any time.
  • No data will be shared externally without your consent.
  • If you are under 13, parental permission is required.

This process aligns with privacy standards including ISO/IEC 27001, ISO/IEC 29100, CCPA/CPRA, COPPA, and the NIST Privacy Framework.

Why It Matters

Why Overshare Shield is important in today’s digital environment.

Prevents Oversharing Before It Happens

Many users share personal details online without realizing the risks. Overshare Shield helps identify those risks before content is posted or shared further.

Reduces Real Cybersecurity Threats

Exposed personal information can contribute to phishing, identity theft, cyberstalking, and social engineering. The platform helps users recognize and reduce that exposure early.

Makes Privacy Easy to Understand

Privacy tools are often too technical for everyday users. Overshare Shield turns complex risk into simple, actionable feedback that users can actually understand and use.

Supports Digital Responsibility

The project encourages better online decision-making by helping users understand how their posts, images, and personal details may affect their security and privacy.
